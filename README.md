# Dumb-NPC-Demo
簡易AI NPC，向綠色方塊提問可得到由GPT提供的回答

## 動作
- 使用wasd操控視角
- 互動對話
  1. 靠近綠色立方體 (距離 < 3)  
  2. 在輸入框輸入問題  
  3. 按下 `Enter` 鍵等待 NPC 回覆

## 實作細節
- 利用Three.js建立虛擬3D空間與簡單的物件互動。
- 使用OpenAI GPT API生成回應內容。

"此版本已停用GPT功能，如需完整功能請在本地端運行並使用有效的OpenAI API金鑰。"
